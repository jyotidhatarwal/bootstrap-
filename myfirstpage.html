A signal jammer is a device used to block all radio transmissions in its range. Signal jammer comes in all kinds of power ranges and sizes depending on the desired application which may vary from individual uses to military purposes. A jamming device transmits signals at the same frequency as the device from which the transmission is to be blocked. However, the signal jammer transmits the signal with much greater power than the targeted device thus disrupting the communication between device(s) and the base station. This is termed as denial-of-service attack. Since communication between device(s) and/or base station is a two-way process the jammer can work in two ways. It can either disrupt the device to base station communication or vice versa.
This project is designed and implemented on cell phone signal jammers using NE555 timer IC. The jammer device generates RF signals in the same frequency in which cell phones work. Jammer jams the transmitted radio signal by the antenna and blocks the cell phone in a virtual area. These days cellular devices work on RF signals and these devices use wireless devices through which they connect to the network.
Mobile jammer is utilized to lessen cell phones from getting or transmitting signals with the base stations. Versatile jammer basically harms cell phones inside the characterized directed zones without making any association other correspondence implies Mobile jammer can be utilized in for all intents and purposes any area, yet are utilized in places where a phone call would be especially troublesome like Temples, Libraries, and Hospitals and so forth.
